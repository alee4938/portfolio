The Fridge

Web / User Interface / Server Hosting

Open in a new tab

This website is a collaborative art site where users can create and edit drawings posted on a shared "fridge", like how children's drawings are often displayed on refrigerators. UI elements were made in Adobe Illustrator and drawings are updated in real time using a server hosted by Firebase.

Made with Adobe Illustrator, Firebase, HTML, CSS, and JavaScript.
